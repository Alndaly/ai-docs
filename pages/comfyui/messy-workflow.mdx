## 应用环境

- ComfyUI https://github.com/comfyanonymous/ComfyUI

## 所需插件

- Anything Everywhere https://github.com/chrisgoringe/cg-use-everywhere

## 步骤

<Steps>
### 安装插件

```shell
cd <path to your local comfyui>/custom_nodes
git clone https://github.com/chrisgoringe/cg-use-everywhere
```

### 启动/重启ComfyUI

```shell
cd <path to your local comfyui>
python main.py
```

### 使用插件

按住键盘上的`Shift`，按住鼠标从任意输出节点拖出，选择`Anything Everywhere?`。基于目标输入设定`title`等参数。

### 运行工作流即可

</Steps>
